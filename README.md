### Техническое задание (ТЗ) для разработки веб-приложения по управлению ценами продуктов на Shopify, привязанными к криптовалютам


---

#### **Общее описание проекта**

Цель проекта — разработка веб-приложения, которое позволяет владельцам магазинов Shopify автоматически обновлять цены на продукты в зависимости от курса выбранных криптовалют. Пользователи смогут создавать и настраивать формулы для расчета цен, выбирать продукты или коллекции, к которым будут применяться эти формулы, и управлять настройками для активации или деактивации обновлений цен.

Привязка к магазину Shopify будет осуществляться через переменные окружения, которые будут подставляться в запросы к API. Авторизация пользователей не требуется, авторизированными будут запросы к серверу Shopify.

#### **Стек**

Frontend: HTML, CSS, JS, Vue.js 3, Vuex
Backend: Node.js
APIs: Shopify Admin REST API, Crypto Compare API
Сборщик проекта: webpack
Развёртывание: Docker 
Контроль версий: GitHub
#### **Функциональные требования**

1. Главная страница:
    
    - Доступ ко всем основным функциям приложения осуществляется через панель управления.
    - Переменные окружения, связанные с магазином Shopify, будут использоваться для авторизации запросов к Shopify Admin REST API.
2. **Создание формул для изменения цены**:
    
    - Пользователь должен иметь возможность создавать формулы для изменения цен продуктов на основе текущего курса выбранных криптовалют.
    - Поддержка различных математических операций (сложение, умножение, деление и т.д.).
    - Визуальный интерфейс для создания и редактирования формул.
3. **Выбор продуктов и коллекций**:
    
    - Пользователь должен иметь возможность выбирать отдельные продукты или коллекции продуктов из своего магазина Shopify, к которым будут применяться созданные формулы.
    - Отображение всех продуктов и коллекций с возможностью фильтрации и поиска.
4. **Создание и управление настройками**:
    
    - Возможность создания настройки, включающей в себя выбранные продукты/коллекции и примененные к ним формулы.
    - Пользователь должен иметь возможность:
        - Активировать/деактивировать настройки.
        - Редактировать настройки (изменять продукты/коллекции и формулы).
        - Удалять настройки.
    - Все настройки должны быть отображены в панели управления с интуитивно понятным интерфейсом.
5. **Интеграция с криптовалютной биржей Binance API**:
    
    - Получение актуальных данных о курсе криптовалют с помощью открытого API.
    - Регулярное обновление данных с учетом выбранного пользователем интервала.
6. **Интеграция с Shopify Admin API**:
    
    - Обновление цен на продукты в магазине Shopify через закрытое Shopify Admin API, используя переменные окружения для связи с магазином.
    - Обработка ошибок и уведомления пользователя в случае неудачных попыток обновления.
7. **Аналитика и история изменения цен**:
    
    - Визуализация истории изменения цен продуктов в зависимости от курса криптовалют.
    - Графики и таблицы для отображения динамики изменений.
8. **Документация и поддержка**:
    
    - Встроенная документация по использованию приложения.
    - Раздел с часто задаваемыми вопросами (FAQ)
9. **Фронтенд интерфейс**:
    
    - Разработка красивого и интуитивно понятного интерфейса, обернутого в лендинг-страницу.
    - Использование современных технологий фронтенда (HTML, CSS, JavaScript, Vue.js, Vuex).
    - Лендинг должен включать информацию о приложении, его возможностях, преимуществах и инструкциях по использованию.
12. **Развёртывание и инфраструктура**:
    
    - Использование Docker для развёртывания приложения.
    - Внутренний сервер для обработки запросов к Shopify Admin API и криптовалютным биржам.
    - Настройка окружения для разработки и продакшена.


#### **Почему именно такой проект**

Данный проект даст студентам возможность почувствовать себя в шкуре веб разработчика, так как это приложение отображает реалии рабочих будней разработчиков. Помимо этого, данный проект смело можно будет добавлять в портфолио, так как он не является чем-то банальным, что обычно Junior разработчики добавляют в своё портфолио. 

А ещё этот проект можно монетизировать, так как в экосистеме Shopify нету подобного приложения, так что, немного, докрутив, его можно будет распространять через Shopify App Store за ежемесячную плату.

Ну и буквально, с технической стороны, проект является лёгким в реализации, даже для новичков.

А ещё, учитывая, что на 3м курсе будут базы данных, можно будет использовать это приложение, что бы "прикрутить" к нему БД и хранить все настройки в БД.
